    Linux Basic Commands

    There are several Linux command we can use within Webstorm terminal. However, we will go through the basics.
    The first command cd is one of the most used. This command allows you to move between directories. The image below shows you that by just typing cd and the name of the directory will take you to it. If you want to one directory back you simply type cd.


    Another common command is ls, which refers to list. There are several ways to use it. For the example below, typing ls will display the content of a directory.


    ls -l will display a more detailed view of the directory. ls -a and ls -s will display a list of directories from the oldest to the newest.



    mkdir is another common command, which is mostly used for creating a directory.

    You could also edit does directories. rm -r deletes a directory and all of the sub directories. rm [filename] will delete the file you typed.


    If you want to rename or move a directory/file to another directory you can do it by using the command mv [filename or directory] [renamed file or moved file directory].


    Dealing With Files

    There is also a command that allows you to create an empty file or edit it. For example, if you use the command touch [name of file], that will create an empty file. For example, if you type touch readme.md, then it will create a document file in the current directory.


    To edit the files you use the command vi [name of file]. A command promt will appear in the terminal where you can write/edit it. To write you press w and to quit writing you press wq. If you want to exit and quit editing that file you can press q to quit.


    A few couple of key words when editing a file through the command prompt. If you press i it means insert. Pressing x means delete a character. Pressing dd means delete a line. Lastly, if you press / it means to search.
