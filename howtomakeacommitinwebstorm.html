    How To Make a Commit in Webstorm


    In this section I will show you how to make a commit in webstorm to eventually be uploaded to github. You should know what a Branch is first so if you havent visited that section, I recommend you do that now. First I want to define what a commit is.


    Commit: A commit, or "revision", is an individual change to a file (or set of files). It's like when you save a file, except with Git, every time you save it creates a unique ID (a.k.a. the "SHA" or "hash") that allows you to keep record of what changes were made when and by who. Commits usually contain a commit message which is a brief description of what changes were made.


    I already created a branch called CommitDefinition which created a copy of my code to work on without disturbing my master code. Now that I entered new information that I like I want to commit it to github. To do this I go to VCS in the menu brr on top and click on Commit.
