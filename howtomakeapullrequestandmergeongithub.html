    How To Make a Pull Request and Merge on Github


    In this section I will show you how make a pull request from webstorm and merge it into the repository on Github. First lets define some terms that we will be using.


    Push: Pushing will commit your changes to the repository on github. You would do this when you want to add you code to the master code in git.


    Pull: A pull is when you taking in the changes made and merging them with your master.
