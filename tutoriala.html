    Autosave/Auto Upload
    In order for your changes to take effect or automatically upload to AFS we must do the following. First you select the Tools bar.


    Then proceed to select Deployment.


    Then click Automatic Uploads. This will allow Webstorm to automatically upload any changes you make to a file or folder. The changes will we displayed once you open the site without having to manually auto-save it.
