    Grunt

    First of all, in order to use Grunt. You must have Node.js installed. Once you already install Node, you can type in the following command in the LOCAL terminal.

    This will install any packages required to run Grunt properly in your system.


    Working with Grunt is much more simpler then manually saving and uploading to AFS. This allows you to get that done with just a click.
    Now, in order to do this you must go to Run then click on the play button Run.... If a windows opens it means that you haven't install Grunt.


    After clicking the play button you will see a small window with some options depending on which .html file you are editing. Just click on it to configure it to that page. Once you do that you don't have to repeat that for that specific .html file. However, if you want to be able to use this feature, you will have to do the same for the others .html files.


    To use this feature you only have to click on the play button which you can find below the Webstorm tabs. If you want to open another .html file directly from the play button, you have to choose it frm the drop down options left side of the play button.
